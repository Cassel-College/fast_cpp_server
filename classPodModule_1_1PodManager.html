详细描述

吊舱管理器（单例模式）

负责统一管理所有吊舱设备：

  • 从 JSON 配置初始化
  • 添加/移除 Pod
  • 查询 Pod
  • 列出所有 Pod
  • 获取状态快照（供 API 层使用）
