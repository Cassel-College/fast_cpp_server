◆ ScanActiveDevices()

ScanResult ScanActiveDevices ( )
static

扫描所有本机网段内的活跃设备

自动调用 GetAllIPs() 获取本机所有 IP / 子网，遍历各网段生成候选 IP 列表，使用原生 ICMP Raw Socket 发送 Echo Request 并等待 Reply。 线程模型为"即时线程"：为每个候选 IP 启动 std::thread，通过信号量 控制最大并发数。

需要 CAP_NET_RAW 权限或 Root。

返回
扫描结果

引用了 ScanResult::error, MyIPTools::GenerateSubnetIPs(), MyIPTools::GetAllIPs(), MyIPTools::HasRawSocketPermission(), MYLOG_DEBUG, MYLOG_ERROR, MYLOG_INFO, MYLOG_WARN, MyIPTools::PingHost(), MyIPTools::s_max_threads, MyIPTools::s_ping_timeout_ms, ScanResult::scan_duration_ms , 以及 ScanResult::total_scanned.
