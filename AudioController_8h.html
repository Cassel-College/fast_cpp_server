详细描述

扬声器模块 REST API 控制器

对外暴露以下接口：

  • GET /v1/audio/config : 查看启动参数（初始化配置）
  • GET /v1/audio/info : 查看指定设备详细信息
  • GET /v1/audio/status : 查看全部设备状态
  • GET /v1/audio/available: 查看当前可用扬声器列表
  • POST /v1/audio/play : 在指定设备上播放音频文件
  • POST /v1/audio/volume : 设置指定设备音量
