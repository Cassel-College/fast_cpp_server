校验文件名合法性

规则：

  • 不得为空
  • 不得包含 '\0' 空字符
  • 不得包含 '\'（反斜杠，Windows 风格路径）
  • 不得以 '/' 开头（绝对路径）
  • 不得包含 ".." 路径组件（路径穿越）
  • 长度不超过 1024 字符
参数
filename待校验的文件名
[out]err_msg校验失败时的错误描述
返回
true 合法，false 非法
